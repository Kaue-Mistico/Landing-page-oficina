# Sobre o projeto

Este projeto foi criado com o intuito de aplicar todos os conhecimentos que adquiri até o momento durante o curso de Formação em Front-end do **Hora de Codar**. Utilizei como base um negócio familiar para a criação do projeto. O objetivo é que o projeto evolua e inclua novas tecnologias conforme minha evolução durante o curso.

Para a criação do layout, me inspirei em outras landing pages do mesmo ramo que encontrei na internet, pensando sempre em transmitir segurança, facilidade na navegação e clareza para o usuário.

## Tecnologias Utilizadas
Front-end:

- HTML5
- CSS3


## Objetivo do projeto

O objetivo deste projeto é aplicar, na prática, os conhecimentos adquiridos em **HTML5 e CSS3**, incluindo:

- Estruturação semântica do HTML
- Organização de layout
- Estilização com CSS
- Boas práticas de código
- Responsividade para os principais dispositivos
